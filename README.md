# 💰 Hisab Kitab

> **AI-Powered Personal Finance & Expense Management Platform**

Hisab Kitab is a modern personal finance web application that helps users automatically track, categorize, and analyze their daily expenses. It provides AI-powered financial insights, budgeting tools, expense reports, and intelligent recommendations to help users build better spending habits.

Designed for **students, professionals, freelancers, and families**, Hisab Kitab transforms scattered transactions into meaningful financial insights.

---


# ✨ Features

### 💳 Automatic Expense Tracking

* Connect your Gmail account to detect payment confirmation emails.
* Automatically extract transaction details.
* Categorize expenses using AI.

### 📝 Manual Transaction Entry

* Add expenses manually.
* Add income manually.
* Support for cash transactions.



### 🤖 AI Expense Categorization

Automatically categorizes expenses into:

* Food
* Shopping
* Travel
* Bills
* Entertainment
* Healthcare
* Education
* Groceries
* Investments
* Others

### 📊 Dashboard

* Total Income
* Total Expenses
* Savings
* Budget Score
* Recent Transactions
* Category Breakdown

### 💰 Budget Management

* Create monthly budgets.
* Edit/Delete budgets.
* Track spending against budget.
* Smart spending indicators.

### 📈 Reports

* Monthly expense summary
* Savings report
* Expense category breakdown
* Financial Health Score
* Monthly comparison
* AI-generated insights

### 🧠 AI Financial Advisor

Receive personalized suggestions like:

* Reduce food expenses.
* Shopping increased compared to last month.
* Stay within your travel budget.
* Improve monthly savings.

### 🔔 Monthly Reports

Users receive monthly summaries including:

* Total Income
* Total Expenses
* Savings
* Budget Utilization
* Financial Score
* AI Recommendations

### 🔍 Smart Search & Filters

* Filter by category
* Filter by month
* Search transactions
* Search merchants

### 📱 Fully Responsive

Works seamlessly across:

* Desktop
* Tablet
* Mobile
